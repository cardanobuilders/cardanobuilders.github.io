# Online Workshop #2

**May 28, 2025** | 35 attendees

The second online workshop in the second milestone of our Cardano Project Catalyst Fund13 proposal, hosted with Cardano India.

This session covered the Mesh SDK in depth: its production-ready TypeScript library, the pure TypeScript and WebAssembly flavors, and the comprehensive documentation system with live demos. Topics included:

- Getting started with the SDK: wallet generation and transaction building
- Change addresses and handling multiple assets in transactions
- The Web3 SDK for user and business onboarding (self-custody, security features)
- SDK roadmap: transaction sponsorship and Free Tokens integration

The session concluded with an open Q&A on SDK capabilities and use cases.
