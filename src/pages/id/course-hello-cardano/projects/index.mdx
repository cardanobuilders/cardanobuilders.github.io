## Panduan Pengumpulan

Untuk mendapatkan lencana **Hello Cardano** Anda, kumpulkan ketiga proyek:

1. **Repositori GitHub**: Push setiap proyek ke repositori GitHub publik (satu repo per proyek atau monorepo dengan direktori terpisah). Setiap proyek harus mencakup:
   - `README.md` dengan instruksi setup
   - Semua kode sumber Aiken dan blueprint yang sudah dikompilasi
   - Semua kode off-chain TypeScript
   - Hasil test (screenshot atau output CI dari `aiken check`)

2. **Hash Transaksi Preprod**: Untuk setiap proyek, sediakan minimal satu hash transaksi preprod yang menunjukkan implementasi Anda yang berfungsi. Tautkan ke [preprod.cardanoscan.io](https://preprod.cardanoscan.io) agar peninjau dapat memverifikasi di on-chain.

3. **Kirim**: Kirimkan tautan repositori dan hash transaksi Anda ke tim kursus untuk ditinjau.
