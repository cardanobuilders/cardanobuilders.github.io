# Workshop Online #2

**28 Mei 2025** | 35 peserta

Workshop online kedua dalam milestone kedua dari proposal Cardano Project Catalyst Fund13 kami, diselenggarakan bersama Cardano India.

Sesi ini membahas Mesh SDK secara mendalam: library TypeScript siap produksi, varian TypeScript murni dan WebAssembly, serta sistem dokumentasi komprehensif dengan demo langsung. Topik yang dibahas meliputi:

- Memulai dengan SDK: pembuatan wallet dan pembangunan transaksi
- Change address dan penanganan multiple asset dalam transaksi
- Web3 SDK untuk onboarding pengguna dan bisnis (self-custody, fitur keamanan)
- Roadmap SDK: sponsorship transaksi dan integrasi Free Tokens

Sesi ini diakhiri dengan tanya jawab terbuka tentang kemampuan dan use case SDK.
