# Hello Cardano

Kursus ini membawa Anda dari nol hingga mampu membangun aplikasi full-stack di Cardano. Melalui 10 pelajaran praktis, Anda akan menulis smart contract dengan [Aiken](https://aiken-lang.org/), membangun transaksi dengan [Mesh SDK](https://meshjs.dev/), dan men-deploy aplikasi yang berfungsi di blockchain Cardano.

Di akhir kursus ini, Anda akan memiliki pengalaman praktis dengan wallet, transaksi, pengembangan smart contract, testing, dan scaling Layer 2.

## Prasyarat

- **TypeScript**: Diperlukan pemahaman dasar TypeScript. Kursus ini berfokus pada konsep Cardano, bukan dasar-dasar bahasa pemrograman.
- **Node.js v24+**: Diperlukan untuk menjalankan kode pelajaran. Kami merekomendasikan penggunaan [nvm](https://github.com/nvm-sh/nvm) untuk mengelola versi Node Anda.
- **Tidak diperlukan pengalaman blockchain**: Setiap pelajaran dibangun di atas pelajaran sebelumnya, dimulai dari dasar.

## Tools dan Sumber Daya

- **[Aiken](https://aiken-lang.org/)**: Bahasa untuk menulis smart contract Cardano yang efisien.
- **[Mesh SDK](https://meshjs.dev/)**: SDK TypeScript untuk membangun transaksi dan berinteraksi dengan blockchain Cardano.
- **[Blockfrost](https://blockfrost.io/)**: Penyedia API blockchain untuk melakukan query ke jaringan Cardano.
- **[Cardano Preprod Testnet Faucet](https://docs.cardano.org/cardano-testnets/tools/faucet)**: Dapatkan test lovelace untuk pengembangan.
