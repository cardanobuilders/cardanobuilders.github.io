# ブロックチェーン理論

このコースは、Web2開発者がブロックチェーンコードの最初の一行を書く前に必要な概念的基盤を提供します。6つのモジュールに整理された15のレッスンを通じて、ブロックチェーンの仕組み、Cardanoが特定の設計選択を行う理由、そしてWeb2の馴染みのある概念がWeb3の同等物にどのようにマッピングされるかを学びます。

このコースを終える頃には、UTXO、コンセンサス、スマートコントラクトバリデーター、ネイティブトークン、DeFiプリミティブ、そしてCardanoガバナンスを理解できるようになります — すべてREST API、データベース、JWT、Gitなど、すでにご存知の概念を通して説明します。

## 前提条件

- **Web開発経験**：HTTP、REST API、データベース、認証の概念に精通していること。
- **ブロックチェーン経験不要**：すべての概念はWeb2のアナロジーとともに基礎から説明します。
- **コーディング不要**：これは理論コースです。ハンズオンコーディングについては、[Hello Cardanoコース](/jp/course-cardano)をご覧ください。
